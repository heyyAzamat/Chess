# ЭДЕМ-ЗЕРО: Протокол Войны

Шахматная игра с нарративом в стиле графического романа. Каждая партия — это не просто доска, а история войны двух фракций за контроль над последним городом.

## Для кого

Для тех, кому скучно играть в шахматы «в вакууме». Для геймеров, которые хотят учиться через истории. Для людей, которые всегда думали, что шахматы — это сухо.

## Что внутри

**Сюжетный режим** - во время игры появляются комикс-панели с диалогами персонажей. Взяли ферзя — сцена с раскрытием маски. Поставили мат — финальная страница с рассветом над городом. Движок сам решает, когда показать следующую главу.

**AI Coach** - после партии тренер разбирает твои ходы через минимакс с альфа-бета отсечением. Показывает мини-доску с позицией в момент ошибки: красное — твой ход, зелёное — лучший. Определяет тактические паттерны (вилка, зевок, незащищённая фигура), делит ошибки по фазам партии и даёт конкретный совет на следующую игру.

**Четыре режима:** обычная игра против бота, сюжетный режим, обучение с подсказками движка, локальный мультиплеер с рейтингом Эло для обоих игроков.

**Экономика:** монеты за партии, ставки, ежедневный бонус, Premium-подписка.

## Почему это ценно

Шахматы обучают мышлению, но большинство людей бросают их на третьей неделе — потому что абстрактно и непонятно, зачем стараться. Нарратив даёт контекст: ты не просто двигаешь коня, ты решаешь, выживет ли Нова. Coach закрывает другую дыру: проигрывать без объяснения причин деморализует, а разбор позиции превращает поражение в урок.
